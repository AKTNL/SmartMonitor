# CpuTools介绍
CpuTools是一个用于在OpenEuler系统中分析CPU性能的工具集。它包括了一系列的工具，可以用于监控、分析CPU的性能问题。
具体能够获得CPU的这几点信息：
1. 总 CPU 使用率 (%)
2. 每个核心的独立使用率列表
3. 逻辑核心数 (线程数)
4. 物理核心数
5. 当前 CPU 频率 (MHz)
6. 上下文切换次数
7. 系统负载 (Load Average)
## 涉及到的psutil工具及其返回的内容
1. psutil.cpu_percent() 用于获取总 CPU 使用率和每个核心的使用率。
```输出示例
总 CPU 使用率 (%): 2.1
每个核心的独立使用率列表: [3.1, 0.0, 7.7, 7.8, 1.5, 1.6, 0.0, 0.0, 6.2, 0.0, 0.0, 0.0, 1.6, 1.6, 3.1, 0.0]
```
2. psutil.cpu_count() 用于获取逻辑和物理核心数。
```输出示例
逻辑核心数 (线程数): 16
物理核心数: 12
```
3. psutil.cpu_freq() 用于获取 CPU 频率。
```输出示例
当前频率: 2600.00MHz
```
4. psutil.cpu_stats() 用于获取上下文切换次数。
```输出示例
上下文切换次数: 242090885
```
5. psutil.getloadavg() 用于获取系统负载。
```输出示例
系统负载 [1分钟, 5分钟, 15分钟]: (0.0, 0.0, 0.0)
```
