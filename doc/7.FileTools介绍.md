# FileTools介绍

`FileTools.py` 模块提供了一系列用于查询文件系统总体情况的工具函数。这些工具旨在帮助用户快速了解指定目录的文件分布、大小概览以及识别占用空间较大的文件。

## 主要功能

### 1. 目录内容概览 (`get_file_example_info` 中的目录概览部分)

此功能用于获取指定目录的整体概览信息，包括：

*   **目录 (Directory)**: 被查询的目录路径（目前硬编码为系统根目录 `/`）。
*   **文件总数 (Total Files)**: 该目录下（不包含子目录）文件的总数量。
*   **子目录总数 (Total Subdirectories)**: 该目录下直接包含的子目录数量。
*   **总大小 (Total Size)**: 该目录下所有文件的总大小。
*   **概览条目 (Overview Entries)**: 列出目录中的前几个文件和子目录的名称及其大小（如果是文件）。

**示例输出 (JSON 格式):**

```json
{
    "状态": "success",
    "目录概览": {
        "目录": "/",
        "文件总数": 1234,
        "子目录总数": 56,
        "总大小": "150.75 GB",
        "概览条目": [
            "bin/                         | DIR",
            "boot/                        | DIR",
            "home/                        | DIR",
            "var/                         | DIR",
            "initrd.img                   | 12.34 MB"
        ]
    },
    "大文件列表": [
        // ... 大文件列表
    ]
}
```

### 2. 查找指定目录中的大文件 (`get_file_example_info` 中的大文件列表部分)

此功能用于在指定目录（目前硬编码为系统根目录 `/`）及其所有子目录中，查找并列出占用空间最大的 N 个文件。对于每个大文件，它会提供：

*   **路径 (Path)**: 文件的完整路径。
*   **大小 (Size)**: 文件的人类可读大小（例如 1.23 GB）。
*   **大小_字节 (Size_Bytes)**: 文件的精确字节大小。
