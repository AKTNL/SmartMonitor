# 目前的Route函数存在的问题

我们的小模型在遇到存在多个关键词的句子的时候，会陷入错误的思路，所以我们现在要做的是构建微调数据集，告诉模型，我们想要的效果是怎样的。

# 目的
1. 让当前的LLM知识中存在对我们的项目的认识
2. 让LLM认识到当用户的input中存在多个关键词的时候定位到system_monitor

# 构建数据集的方法
利用我们事先定义好的提示词让AI根据我们的Keywords生成指定格式的数据

提示词:
