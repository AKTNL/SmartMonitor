# DiskTools介绍

`DiskTools.py` 模块提供了一系列用于获取系统磁盘使用情况和性能统计的工具函数。这些工具能够帮助用户全面了解磁盘的健康状况和资源分配。

## 主要功能

### 1. 获取磁盘分区信息 (`get_disk_partitions_info`)

此功能用于列出系统中所有物理和逻辑磁盘分区的信息。对于每个分区，它会提供以下详细数据：

*   **设备 (Device)**: 磁盘设备的名称，例如 `/dev/sda1` 或 `C:`。
*   **挂载点 (Mountpoint)**: 分区在文件系统中的挂载路径，例如 `/` 或 `/home`。
*   **文件系统类型 (Filesystem Type)**: 分区所使用的文件系统格式，例如 `ext4`、`NTFS`。
*   **总容量 (Total Capacity)**: 分区的总存储空间。
*   **已用容量 (Used Capacity)**: 分区已被占用的存储空间。
*   **空闲容量 (Free Capacity)**: 分区剩余可用的存储空间。
*   **使用率 (Usage Percentage)**: 分区存储空间的使用百分比。

**示例输出 (JSON 格式):**

```json
{
    "状态": "success",
    "分区信息": [
        {
            "设备": "/dev/sda1",
            "挂载点": "/",
            "文件系统类型": "ext4",
            "总容量": "100.00 GB",
            "已用容量": "30.00 GB",
            "空闲容量": "70.00 GB",
            "使用率": "30%"
        },
        // ... 其他分区
    ],
    "IO统计": {
        "读取次数": 12345,
        "写入次数": 67890,
        "读取字节": "1.23 GB",
        "写入字节": "4.56 GB"
    }
}
```

### 2. 获取磁盘 I/O 统计 (`get_disk_io_counters`)

此功能提供系统级别的磁盘输入/输出操作统计，包括：

*   **读取次数 (Read Count)**: 自系统启动以来，磁盘读取操作的总次数。
*   **写入次数 (Write Count)**: 自系统启动以来，磁盘写入操作的总次数。
*   **读取字节 (Read Bytes)**: 自系统启动以来，从磁盘读取的总字节数。
*   **写入字节 (Write Bytes)**: 自系统启动以来，写入磁盘的总字节数。

**示例输出 (JSON 格式):**

```json
{
    "状态": "success",
    "分区信息": [
        // ... 分区信息
    ],
    "IO统计": {
        "读取次数": 12345,
        "写入次数": 67890,
        "读取字节": "1.23 GB",
        "写入字节": "4.56 GB"
    }
}
```
