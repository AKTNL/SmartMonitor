# NetworkTools介绍

`NetworkTools.py` 模块提供了一系列用于获取系统网络使用情况和性能统计的工具函数。这些工具能够帮助用户全面了解网络接口的状态、流量、连接以及实时速度。

## 主要功能

### 1. 网络流量统计 (累计) (`get_network_example_info` 中的累计流量统计部分)

此功能用于获取自系统启动以来，所有网络接口的累计流量统计信息，包括：

*   **发送字节 (Bytes Sent)**: 累计发送的字节数。
*   **接收字节 (Bytes Received)**: 累计接收的字节数。
*   **发送包数 (Packets Sent)**: 累计发送的数据包数量。
*   **接收包数 (Packets Received)**: 累计接收的数据包数量。
*   **发送错误数 (Send Errors)**: 发送数据时发生的错误数量。
*   **接收错误数 (Receive Errors)**: 接收数据时发生的错误数量。
*   **丢弃包数 (Dropped Packets)**: 累计丢弃的数据包数量。

**示例输出 (JSON 格式):**

```json
{
    "状态": "success",
    "累计流量统计": {
        "发送字节": "1.23 GB",
        "接收字节": "4.56 GB",
        "发送包数": 123456,
        "接收包数": 789012,
        "发送错误数": 0,
        "接收错误数": 0,
        "丢弃包数": 0
    },
    "实时网速": {
        // ... 实时网速
    },
    "网络接口地址": [
        // ... 网络接口地址
    ],
    "网络接口状态": [
        // ... 网络接口状态
    ],
    "活跃连接": [
        // ... 活跃连接
    ]
}
```

### 2. 实时网速 (`get_network_example_info` 中的实时网速部分)

此功能用于计算并显示在指定时间间隔内（默认为 1.0 秒）的网络上传和下载速度，提供：

*   **发送速度 (Upload Speed)**: 实时上传速度。
*   **接收速度 (Download Speed)**: 实时下载速度。

**示例输出 (JSON 格式):**

```json
{
    "状态": "success",
    "累计流量统计": {
        // ... 累计流量统计
    },
    "实时网速": {
        "发送速度": "10.24 KB/s",
        "接收速度": "1.50 MB/s"
    },
    "网络接口地址": [
        // ... 网络接口地址
    ],
    "网络接口状态": [
        // ... 网络接口状态
    ],
    "活跃连接": [
        // ... 活跃连接
    ]
}
```

### 3. 网络接口地址 (`get_network_example_info` 中的网络接口地址部分)

此功能用于列出系统中所有网络接口的 IP 地址信息，包括 IPv4 和 IPv6 地址，以及对应的子网掩码和广播地址：

*   **接口 (Interface)**: 网络接口的名称，例如 `eth0` 或 `Wi-Fi`。
*   **地址类型 (Address Type)**: IP 地址的类型，如 `IPv4` 或 `IPv6`。
*   **地址 (Address)**: 接口的 IP 地址。
*   **子网掩码 (Netmask)**: 接口的子网掩码。
*   **广播地址 (Broadcast Address)**: 接口的广播地址 (仅 IPv4)。

**示例输出 (JSON 格式):**

```json
{
    "状态": "success",
    "累计流量统计": {
        // ... 累计流量统计
    },
    "实时网速": {
        // ... 实时网速
    },
    "网络接口地址": [
        {
            "接口": "Ethernet",
            "地址类型": "IPv4",
            "地址": "192.168.1.100",
            "子网掩码": "255.255.255.0",
            "广播地址": "192.168.1.255"
        },
        {
            "接口": "Loopback",
            "地址类型": "IPv6",
            "地址": "::1",
            "子网掩码": "ffff:ffff:ffff:ffff:ffff:ffff:ffff:ffff"
        }
    ],
    "网络接口状态": [
        // ... 网络接口状态
    ],
    "活跃连接": [
        // ... 活跃连接
    ]
}
```

### 4. 网络接口状态 (`get_network_example_info` 中的网络接口状态部分)

此功能提供每个网络接口的当前状态信息，包括：

*   **接口 (Interface)**: 网络接口的名称。
*   **是否开启 (Is Up)**: 接口是否处于活动状态 (True/False)。
*   **是否全双工 (Is Full Duplex)**: 接口是否工作在全双工模式 (True/False)。
*   **速度 (Speed)**: 接口的连接速度，单位 Mbps。
*   **MTU (Maximum Transmission Unit)**: 接口的最大传输单元。

**示例输出 (JSON 格式):**

```json
{
    "状态": "success",
    "累计流量统计": {
        // ... 累计流量统计
    },
    "实时网速": {
        // ... 实时网速
    },
    "网络接口地址": [
        // ... 网络接口地址
    ],
    "网络接口状态": [
        {
            "接口": "Ethernet",
            "是否开启": true,
            "是否全双工": true,
            "速度": "1000 Mbps",
            "MTU": 1500
        }
    ],
    "活跃连接": [
        // ... 活跃连接
    ]
}
```

### 5. 活跃网络连接 (`get_network_example_info` 中的活跃连接部分)

此功能用于列出当前系统中的活跃网络连接（仅显示 TCP/UDP 连接，且限制显示前 5 个），提供：

*   **进程ID (Process ID)**: 拥有该连接的进程 ID。
*   **本地地址 (Local Address)**: 本地 IP 地址和端口。
*   **远程地址 (Remote Address)**: 远程 IP 地址和端口。
*   **状态 (Status)**: 连接的当前状态（例如 `ESTABLISHED`, `LISTEN`）。
*   **类型 (Type)**: 连接的类型（例如 `SOCK_STREAM` 代表 TCP）。

**示例输出 (JSON 格式):**

```json
{
    "状态": "success",
    "累计流量统计": {
        // ... 累计流量统计
    },
    "实时网速": {
        // ... 实时网速
    },
    "网络接口地址": [
        // ... 网络接口地址
    ],
    "网络接口状态": [
        // ... 网络接口状态
    ],
    "活跃连接": [
        {
            "进程ID": 1234,
            "本地地址": "192.168.1.100:54321",
            "远程地址": "10.0.0.5:80",
            "状态": "ESTABLISHED",
            "类型": "SOCK_STREAM"
        }
    ]
}
