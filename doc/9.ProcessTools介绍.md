# ProcessTools介绍

`ProcessTools.py` 模块提供了一系列用于获取系统进程总体情况的工具函数。这些工具旨在帮助用户快速了解当前系统运行的进程状态、资源占用情况以及识别高负载进程。

## 主要功能

### 1. 高资源占用进程 (Top 5) (`get_process_example_info` 中的高资源占用进程部分)

此功能用于识别并列出当前系统中 CPU 和内存资源占用最高的 5 个进程。对于每个高资源占用进程，它会提供：

*   **PID (Process ID)**: 进程的唯一标识符。
*   **名称 (Name)**: 进程的名称。
*   **用户 (User)**: 运行该进程的用户。
*   **CPU使用率 (CPU Usage)**: 进程在指定时间间隔内的 CPU 使用百分比。
*   **内存使用 (Memory Usage)**: 进程当前占用的物理内存大小。

**示例输出 (JSON 格式):**

```json
{
    "状态": "success",
    "高资源占用进程": [
        {
            "PID": 1234,
            "名称": "python",
            "用户": "user",
            "CPU使用率": "15.23%",
            "内存使用": "150.50 MB"
        },
        // ... 其他高资源占用进程
    ],
    "进程概览": [
        // ... 进程概览
    ],
    "进程总数": 150
}
```

### 2. 进程概览 (前20个) (`get_process_example_info` 中的进程概览部分)

此功能提供当前系统运行的前 20 个进程的基本信息概览，包括：

*   **PID (Process ID)**: 进程的唯一标识符。
*   **名称 (Name)**: 进程的名称。
*   **状态 (Status)**: 进程的当前运行状态（例如 `running`, `sleeping`）。
*   **内存使用 (Memory Usage)**: 进程当前占用的物理内存大小。

**示例输出 (JSON 格式):**

```json
{
    "状态": "success",
    "高资源占用进程": [
        // ... 高资源占用进程
    ],
    "进程概览": [
        {
            "PID": 5678,
            "名称": "chrome",
            "状态": "running",
            "内存使用": "200.75 MB"
        },
        // ... 其他进程概览
    ],
    "进程总数": 150
}
```

### 3. 进程总数 (`get_process_example_info` 中的进程总数部分)

此功能返回当前系统正在运行的进程总数量。

**示例输出 (JSON 格式):**

```json
{
    "状态": "success",
    "高资源占用进程": [
        // ... 高资源占用进程
    ],
    "进程概览": [
        // ... 进程概览
    ],
    "进程总数": 150
}
```
